What you get

Built for people who actually read notebooks

◧

Drag, drop, batch

Drop a folder's worth of notebooks at once. Reorder, remove, or clear the queue before you convert anything.

◨

Syntax-highlighted code

Code cells keep keyword, string, and comment colouring so output stays legible, not a wall of grey text.

▤

Real markdown rendering

Headings, bold, italics, lists, and inline code render properly instead of showing raw **asterisks**.

◫

Document-grade layout

Every PDF gets a title page, generation date, consistent margins, and numbered pages — ready to send or archive.

⟳

Live progress, per file

Each file shows its own progress bar and status, so a large batch never feels like a black box.

◑

Dark & light themes

Pick the theme that matches your editor. Your choice is remembered next time you open the tool.

Your notebooks never leave your device.

Every conversion runs inside your browser tab using local JavaScript. Files are read with the FileReader API, parsed in memory, and rendered straight into a PDF — nothing is uploaded, logged, or transmitted anywhere. Close the tab and it's as if it never happened.

  • No server. There is nothing to upload to in the first place.
  • No accounts. Nothing to sign up for, nothing tied to your identity.
  • No tracking. No analytics pixels reading your file names or contents.
